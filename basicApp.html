Git Command Line Practice

Preperation

cd.. - move to previous location in directory
cd 'nameOfLocation' - moves to specified location in directory
mkdir 'name' - create folder in directroy location

Unstaged

git add --all - stages all files
git add 'nameOfFil' - stages specified file
git init - prepares files for version control
git branch 'name' - creates branch with name
git checkout nameOfBranch - move into branch

Staged

git Commit -m "Messege here"

Commit

git push 'URl of repo' 'nameOfBranch' - only need to do once
git push - can be used after using URL push
git pull - pulls from most recent repo
git pull origin 'nameOfBranch' - pulls most recent from specified branch
